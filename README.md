
## 📥 Installation

1. Go to the [Releases](https://github.com/SumeetNayak-NITRR/trackeros/releases/latest) page.
2. Under "Assets", download the latest **`TrackerOS Setup 1.0.0.exe`** file.
3. Double-click the downloaded `.exe` to install.
   > ⚠️ **Note on Windows SmartScreen**: Because TrackerOS is an independent open-source project, it isn't signed with a paid enterprise certificate. You might see a blue "Windows protected your PC" warning. Simply click **More info** -> **Run anyway** to proceed safely.
4. **You're all set!** TrackerOS will launch and guide you through your first Daily Setup. Moving forward, it will automatically launch quietly in the background whenever you turn on your laptop.

### ⌨️ Default Keyboard Shortcut
- **Toggle TrackerOS:** `Ctrl` + `Shift` + `W`
